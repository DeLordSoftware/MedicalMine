*In Netbeans
Tools-> library -> New Library (give name)
click Add jar folder
Navigate to where javafx jar are located. (javafx\lib\)
Add all the jar files

*To activate javafx in project
On IDE, right click on java project name and select properties
Select Categories: Library and then click on plus (+) of the classpath
Select the library created in previous step
Then select the "run" tab 
click on plus (+) of the modulepath and repeat previous step 

Select Categories: Run
add the following text in the VM Options text box.
"--module-path C:\javafx-sdk-11.0.2\lib --add-modules=javafx.controls,javafx.fxml"
